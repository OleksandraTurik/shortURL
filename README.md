# Short URL

## How to start server

### Install depends

- `cd api`
- `yarn` or `npm i`

### Create .env

- `cp dist.env .env` or create manually
- add your config

### Run

- `yarn dev` or `npm run dev`

## How to start client

❗️`yarn` is required

### Install depends

- `cd ui`
- `yarn`

### Run

- `yarn dev`
